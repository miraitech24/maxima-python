# 課題：微分幾何学と時空の歪みの解析（連成問題）

## 1. 課題概要

本課題では、一般相対性理論において重力を記述する「時空のゆがみ」を数学的に扱います 。

具体的には、2次元の曲面（球面や擬球面）を想定し、その曲がり具合を示す「リッチ曲率」を計算し、その曲がりを平滑化する「リッチフロー」の挙動を解析します 。

## 2. 数式仕様

扱う数学的対象は以下の通りです。

1. **計量テンソル ($g_{ij}$)**: 空間の「ものさし」を定義します。

2. **リッチテンソル ($R_{ij}$)**: 空間がどのように曲がっているかを記述します 。

3. リッチフロー方程式:
   
   $\frac{\partial g_{ij}}{\partial t} = -2 R_{ij}$
   
   これは図形の歪みを熱が伝わるように平滑化していく方程式です 。

## 3. 実施手順

### Step 1: Maximaによる代数計算

以下の処理を行う Maxima スクリプト `geometry.mac` を作成せよ。

- 2次元の極座標系における計量テンソル $g_{ij}$ を定義する。

- クリストッフェル記号およびリッチテンソル $R_{ij}$ を代数的に導出する。

- 計算結果（リッチテンソルの成分）を Python が読み込める形式（リストまたは式）でファイル `result_tensor.txt` に出力する。

### Step 2: Pythonによる数値解析と可視化

`result_tensor.txt` を読み込み、以下の処理を行う Python スクリプト `flow_sim.py` を作成せよ。

- **リッチフローの更新**: 導出された $R_{ij}$ を用いて、微小時間 $\Delta t$ ごとに $g_{ij}$ を更新する。

- **曲面の可視化**: 空間が平滑化（ポアンカレ予想の証明のようにシワが伸びる過程）される様子を Matplotlib 等でグラフ化する 。

## 4. 実行方法

Bash

```
# Maximaで代数計算を実行
maxima -b geometry.mac

# Pythonで結果をインポートしてシミュレーション実行
python flow_sim.py
```
